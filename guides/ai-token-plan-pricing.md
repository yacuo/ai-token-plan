# AI Token Plan 套餐价格目录：订阅、额度、重置、优惠综合对比

这个专题整理适合个人、开发者和小团队关注的 AI Token Plan。重点不是只看月费，而是同时看模型能力、额度、额度重置、免费额度、优惠入口、官网入口和是否能用于 Claude Code / Cursor 等工具。

## 选套餐时要看什么

| 维度 | 判断方式 |
|---|---|
| 价格 | 不只看月费，还要看高级模型是否额外扣额度 |
| 额度 | 看总额度、日额度、月额度、上下文长度和是否区分模型 |
| 重置 | 看额度按天、按月还是按周期重置 |
| 优惠 | 看学生、年付、首月、活动码、渠道优惠是否存在 |
| 接入 | 看能否用于 API、Claude Code、Cursor、Agent、插件或 MCP 工具 |
| 国内可用性 | 看支付、访问、实名认证、控制台稳定性 |

## 推荐场景

### 个人日常使用

优先看 Claude、Gemini、Kimi。重点是聊天体验、额度是否够用、长上下文和多端使用。

### AI 编程和 Agent

优先看 Claude、Kimi、MiniMax、GLM。重点看代码能力、工具调用、Claude Code 接入、Cursor 接入和额度消耗速度。

### 国内团队落地

优先看 Kimi、MiniMax、GLM、Qwen、Doubao。重点看人民币支付、企业管理、发票、国内访问和合规。
